### Laravel Auth Microservice - PKCE OAuth2 Flow

This microservice acts as the central Authorization Server and Identity Provider for the entire microservice architecture.
It is responsible for managing user accounts, credentials, and issuing secure access tokens based on the OAuth 2.0 
Authorization Code Grant.
